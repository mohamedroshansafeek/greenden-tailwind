
🌱 **Project Name**: GREENDEN – Plant & Flower Seller Website

---

🧱 **1. Technology Used**:

* **HTML**: For structuring the content and layout of the web pages.
* **Tailwind CSS**: For fast and modern styling using utility-first classes.
* ✅ No JavaScript or frameworks used – purely built with HTML and Tailwind CSS.

---

📱 **2. Responsive Design**:

* The website is **fully responsive**, meaning it works well on:

  * Mobile phones 📱
  * Tablets 📱
  * Laptops 💻
  * Desktop screens 🖥️
* Used **Tailwind’s responsive utilities** (like `sm:`, `md:`, `lg:`) to adjust layout, font sizes, and images for different devices.

---

🎨 **3. Design and Layout**:

* **Hero Section** with a beautiful banner and a call-to-action button to attract visitors.
* **Product Section** that displays various categories of plants and flowers.
* **About Section** to describe GREENDEN’s purpose and passion for nature.
* **Contact or Footer Section** with useful links, contact info, and social media icons.
* Used **consistent colors**, rounded corners, and spacing for a neat, fresh look.

---

🛒 **4. User Experience (UX)**:

* Clean and simple navigation bar at the top.
* Clear headings and organized content blocks.
* Visual hierarchy maintained using Tailwind classes like `text-xl`, `font-bold`, `p-4`, etc.
* Easy to browse and shop (even if there’s no actual cart functionality yet).

---

📂 **5. Project Goals**:

* To **practice responsive design** using Tailwind CSS.
* To create a **realistic, professional-looking website** for plant and flower selling.
* To **improve UI design skills** using utility-first CSS approach.
